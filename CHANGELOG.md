# Changelog

All notable changes to `polymarket-laravel` will be documented in this file.

## 1.0.0 - 2024-12-12

### Added

- Initial release
- Service provider with Laravel auto-discovery
- Facade support for static access to Polymarket client
- Configuration publishing with environment variable support
- Install command (`php artisan polymarket:install`) for guided setup
- Support for both Gamma API (market data) and CLOB API (trading)
- Singleton client binding for optimal performance
- Comprehensive test suite with Pest
- PHPStan level 9 static analysis
- Laravel Pint code formatting
- GitHub Actions workflows for tests and code style
- Support for Laravel 10.x and 11.x
- Support for PHP 8.1, 8.2, and 8.3
- Comprehensive documentation and usage examples
